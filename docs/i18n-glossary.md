# i18n Glossary — QS Technology

## Brand Voice

- **Tone**: Formal, B2B technical. Avoid colloquial language.
- **Audience**: Industrial engineers, plant managers, automation integrators.
- **Style**: Concise, specification-driven. Prefer noun phrases over verb phrases in labels.

## Notes for Translators

1. Always use "Controller" (not "Regulator") for "Bộ điều khiển".
2. "Dịch vụ" = "Services" in navigation context; "Service" (singular) in detail page context.
3. Keep technical acronyms (PLC, HMI, CNC, EtherCAT) unchanged.
4. Spindle refers to the rotating cutting tool holder — never translate to "main axis".
5. "Made in Vietnam" is a brand statement — keep as-is in English.
